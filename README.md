Kindle 即時翻譯 - Chrome 擴充功能
專為 Kindle Cloud Reader（read.amazon.com）設計的即時翻譯工具。由於 Kindle Cloud Reader 使用特殊的 Canvas 渲染技術，市面上所有翻譯擴充功能都無法直接讀取書本文字。
本工具採用 AI 視覺辨識技術，自動截取當前書本頁面，透過 Claude AI 辨識並翻譯成繁體中文，翻譯結果即時顯示在右側面板，讓你一邊看原文一邊對照中文，完全不需要手動複製貼上。
主要功能

一鍵截圖翻譯當前頁面
繁體中文顯示於右側面板
支援自動翻譯每頁（翻頁自動觸發）
翻譯費用極低，一整本書約台幣 15 元 (API)

使用需求

Claude API Key（前往 console.anthropic.com 申請）
Chrome 瀏覽器
